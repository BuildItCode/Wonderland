## GoRouter — All New Projects

```dart
// router.dart — single source of truth
@riverpod
GoRouter router(RouterRef ref) => GoRouter(
  routes: $appRoutes,   // generated by go_router_builder
  redirect: (ctx, state) {
    final isAuth = ref.read(authProvider).isAuthenticated;
    final isLogin = state.uri.path == '/login';
    if (!isAuth && !isLogin) return '/login';
    if (isAuth && isLogin) return '/';
    return null;
  },
);

// Typed route — go_router_builder annotation
@TypedGoRoute<UserDetailRoute>(path: '/users/:userId')
class UserDetailRoute extends GoRouteData {
  const UserDetailRoute({required this.userId});
  final String userId;

  @override
  Widget build(BuildContext context, GoRouterState state) =>
      UserDetailScreen(userId: userId);
}
```

**Rules:**
- `go_router_builder` for compile-time typed routes — no raw `context.go('/path')` strings
- Auth redirect centralised in the router `redirect` callback — not in individual screens
- `ShellRoute` for persistent navigation shells (bottom nav, side drawer)
- Route constants are typed `GoRouteData` subclasses
- Deep links declared in router config and validated on entry

## Navigation 2 (flutter/skills default — without go_router_builder)

```dart
// Handles routing, navigation, and deep linking in a Flutter application.
// flutter/skills: Use when moving between screens or setting up URL-based navigation.

final router = GoRouter(
  routes: [
    GoRoute(
      path: '/',
      builder: (context, state) => const HomeScreen(),
      routes: [
        GoRoute(
          path: 'users/:id',
          builder: (context, state) => UserDetailScreen(
            userId: state.pathParameters['id']!,
          ),
        ),
      ],
    ),
  ],
);
```

## Common Patterns

### Deep Links
```dart
GoRouter(
  routes: [...],
  // Declare deep link handling at router level
  // Validate parameters before acting on them
)
```

### Auth Guard
```dart
redirect: (context, state) {
  final authenticated = ref.read(authProvider).isAuthenticated;
  final goingToLogin = state.matchedLocation == '/login';
  if (!authenticated && !goingToLogin) return '/login';
  if (authenticated && goingToLogin) return '/';
  return null;
},
```

### Returning Results
```dart
// Push and await result
final result = await context.push<bool>('/confirm');
if (result == true) { /* confirmed */ }
```
